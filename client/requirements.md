## Packages
framer-motion | Page transitions, gauge animations, and list animations
date-fns | Formatting timestamps and elapsed time
clsx | Conditional Tailwind class merging
tailwind-merge | Merging Tailwind classes safely

## Notes
- App relies on Polymarket Gamma API and CLOB API. Fallback mock data is included in case of CORS or network failures to ensure the dashboard remains functional and visually verifiable.
- Using Google Fonts: JetBrains Mono (data/monospace) and IBM Plex Sans (body).
